Array Practice ->


### 1. Array Initialization and Basic Operations
**Question:**
Create an array of your favorite fruits. Perform the following operations:
1. Add a new fruit to the end of the array.
2. Remove the last fruit from the array.
3. Add a new fruit to the beginning of the array.
4. Remove the first fruit from the array.

### 2. Array Iteration
**Question:**
Write a function that takes an array of numbers and prints each number to the console.

### 3. Array Mapping
**Question:**
Given an array of numbers, return a new array where each number is squared.

### 4. Array Filtering
**Question:**
Write a function that takes an array of numbers and returns a new array containing only the even numbers.

### 5. Array Reducing
**Question:**
Create a function that takes an array of numbers and returns the sum of all the numbers.

### 6. Array Finding
**Question:**
Write a function that takes an array of numbers and a target number, and returns the first index where the target number is found. If the target number is not found, return -1.

### 7. Array Sorting
**Question:**
Given an array of numbers, write a function that returns a new array with the numbers sorted in ascending order.

### 8. Array Concatenation
**Question:**
Write a function that takes two arrays and returns a new array that is the concatenation of the two arrays.

### 9. Array Slicing
**Question:**
Given an array of numbers, write a function that returns a new array containing only the first three elements of the original array.

### 10. Array Splicing
**Question:**
Write a function that takes an array of numbers and removes the second and third elements, returning the modified array.
